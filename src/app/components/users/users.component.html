As requested for the coding challenge, here I present a simple functional app developed with Angular 8, Firestore RealTime Database, Firebase Hosting for structure, while using both Bootstrap 4 and Angular Material for certain styling requirements.

This app retrieves the users information stored in the Firebase RealTime Database, to later populate a table that keeps data organiced and enables the user to filter and sort the records at will. To Sort the rows, just click on the column header, it will toggle between ascending to descending. To Filter the rows, just click on the hamburguer menu at the column header and the search menu will pop-up.

If you want to add a new user record, just click at the Add user button to display the form and proceed with filling the text inputs.
